### Troubleshooting

#### Button Not Appearing?
- Make sure you're logged in
- Refresh the page
- Check if you're on login/register pages (button hidden there)

#### Mobile View Not Loading?
- Check browser console for errors
- Ensure JavaScript is enabled
- Try a different browser
- Clear browser cache

#### Iframe Issues?
- Some features may be limited in iframe mode
- Authentication state is shared
- Navigation works normally
